# raspberrypi-setup

raspbian and similar distros
```
cd
git clone https://github.com/chrisbernal/raspberrypi-setup.git setup
sudo sh setup/setup.sh
```

minibian requires git and sudo to be installed
```
apt-get update -y
apt-get install git sudo -y
cd
git clone https://github.com/chrisbernal/raspberrypi-setup.git setup
sudo sh setup/setup.sh
```
